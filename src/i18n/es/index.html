🚀 Super rápido

Parcel utiliza procesos de trabajo para habilitar la compilación multinúcleo, y tiene una caché del sistema de archivos para reconstrucciones rápidas incluso después de un reinicio.

📦 Empaqueta cualquier cosa

Parcel tiene soporte para empaquetar JS, CSS, HTML, archivos y más, no se necesitan complementos.

🐠 Transformaciones

El código se transforma automáticamente usando Babel, PostCSS, and PostHTML cuando sea necesario - incluso node_modules.

✂️ Code Splitting

Usando la sintaxis de importación dinámica import(), Parcel empaqueta tu web en módulos automáticamente para cargar sólo lo necesario.

🔥 Hot module replacement

Parcel actualiza automáticamente los módulos en el navegador cada vez que haces un cambio durante el desarrollo, no necesita ningúna configuración

🚨 Registro de errores

Parcel muestra bloques de código cuando encuentra un error para ayudarte a encontrar el error fácilmente.

Hola Mundo

Comienza con el archivo HTML de entrada de tu aplicación. Parcel encuentra todas las dependencias partiendo de ese archivo.

✏️ index.html 

<html>
<body>
  <script src="./index.js"></script>
</body>
</html>

🛠 index.js 

// importa otro componente
import main from './main';

main();

🛠 main.js 

// importa un módulo CSS
import classes from './main.css';

export default () => {
  console.log(classes.main);
};

💅 main.css 

.main {
  /* Referencia un archivo de imágen */
  background: url('./images/background.png');
  color: red;
}

Solo ejecuta parcel index.html para lanzar un servidor de desarrollo. Importa JavaScript, CSS, images, y más! 👌
