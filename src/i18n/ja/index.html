🚀 とても速いバンドル

Parcelはワーカープロセスを利用してマルチコアコンパイルを可能にし、再起動後のビルドを高速化するためのファイルシステムキャッシュも備えています。

📦 全てのアセットをバンドル

Parcel はインストール直後からJS、CSS、HTML、ファイルアセットなどをサポートしています。プラグインは必要ありません。

🐠 自動で変換

コードはBabel、PostCSS、PostHTMLを使って自動で変換されます。必要であればnode_modulesも。

✂️ 設定なしのコード分割

動的な import() 記法で、Parcelは出力結果を分割します。最初の読み込みに必要なファイルだけをバンドルすることができます。

🔥 ホットモジュールリプレースメント

あなたが開発中に変更を加えると、Parcelは自動的にブラウザに読み込まれたモジュールを更新します。設定は必要ありません。

🚨 分かりやすいエラーログ

Parcelはエラーが発生した場合、シンタックスハイライトされたコードを出力する事で問題を特定しやすくします。

Hello World

HTMLファイルからはじめましょう。Parcelはそこから依存関係にしたがって、アプリ全体をビルドします。

✏️ index.html

<html>
<body>
  <script src="./index.js"></script>
</body>
</html>

🛠 index.js

// 別のコンポーネントをインポート
import main from './main';

main();

🛠 main.js

// CSS moduleをインポート
import classes from './main.css';

export default () => {
  console.log(classes.main);
};

💅 main.css

.main {
  /* 画像ファイルを参照します */
  background: url('./images/background.png');
  color: red;
}

parcel index.htmlを実行するだけで開発用サーバーが起動します。JavaScript、CSS、画像などをインポートすることが出来ます！👌
